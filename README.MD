# GBlake

GBlake — полнофункциональное веб-приложение (социальная сеть / платформа для блогинга), полностью написанное на TypeScript. Проект предоставляет возможности для регистрации пользователей, создания постов, формирования ленты новостей, поиска и администрирования.

## Основные возможности

- **Аутентификация и безопасность**: Регистрация, вход в систему (JWT токены + bcrypt для хеширования паролей).
- **Посты и персональная лента**: Создание новых публикаций, загрузка медиафайлов, просмотр ленты.
- **Поиск**: Удобный поиск по контенту и пользователям.
- **Админ-панель**: Инструменты для модерации платформы.
- **Адаптивный UI**: Современный и отзывчивый интерфейс благодаря Tailwind CSS + всплывающие уведомления (react-hot-toast).

## Технологический стек

### Backend (API)

- **Runtime**: Node.js
- **Фреймворк**: Express.js
- **Язык**: TypeScript
- **База данных**: MongoDB (через ORM Mongoose)
- **Валидация**: express-validator
- **Работа с файлами**: Multer

### Frontend (Клиент)

- **Библиотека**: React 19
- **Язык**: TypeScript
- **Стилизация**: Tailwind CSS
- **Роутинг**: React Router DOM
- **HTTP-клиент**: Axios

## Установка и запуск (Local Development)

Для локального запуска убедитесь, что у вас установлены **Node.js** и **MongoDB**.

1. **Клонируйте репозиторий:**

    ```bash
    git clone https://github.com/blake22020/Gblake.git
    cd Gblake
    ```

2. **Настройка переменных окружения:**
   В корне проекта создайте файл `.env`. Пример необходимой конфигурации:

    ```env
    PORT=3000
    MONGODB_URL=mongodb://localhost:27017/gblake
    JWT_SECRET=твой_секретный_ключ
    ```

3. **Сборка и запуск:**

    _Бэкенд (корень проекта):_

    ```bash
    npm install
    npm run dev
    ```

    _Фронтенд (в отдельной вкладке терминала):_

    ```bash
    cd frontend
    npm install
    npm start
    ```

## Сборка для продакшена (Production)

Сервер настроен на раздачу статики React из папки `frontend/build`.

1. Соберите фронтенд-часть:

    ```bash
    cd frontend
    npm run build
    ```

2. Соберите бэкенд и запустите:
    ```bash
    cd ..
    npm run build
    npm start
    ```
    После этого приложение будет доступно по адресу `http://localhost:3000` (если порт не переопределен в `.env`).

## Скрипты

- `npm run dev` — запуск бэкенда с горячей перезагрузкой (ts-node-dev).
- `npm run build` — компиляция TypeScript кода бэкенда в папку `dist`.
- `npm start` — запуск скомпилированного кода (`node dist/server.ts`).
- `npm run clear-db` — очистка базы данных (для дебага).
